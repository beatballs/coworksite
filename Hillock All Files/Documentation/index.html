A) HTML Structure - top

Hillock is a modernistic one page HTML Template powered by Bootstrap for creative agencies, corporate company, small business or as personal portfolio for a freelancer. Perfect for showing off your latest projects/works in all their glory with the responsive portfolio grid style.
